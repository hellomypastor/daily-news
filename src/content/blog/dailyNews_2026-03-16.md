### [Scryer：AI代理的共享架构建模工具](https://github.com/aklos/scryer)

来源：Hacker News - Newest: "AI"

发布时间：2026-03-16 21:00:07

Scryer 是一款面向 AI 代理的可视化 C4 架构建模工具，允许用户与 AI 代理共享同一模型。其核心功能是通过 MCP 协议让代理读取、修改并生成代码。支持 Claude Code 和 Codex，适用于代码生成与架构同步的场景。其 UI 使用 React 和 TypeScript 构建，后端基于 Rust，适合开发者进行架构管理与团队协作。
